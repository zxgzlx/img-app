# 资源整理工具

> 学习rust的一些开源tauri的使用

# 小工具目前目标

使用[Tauri](https://tauri.app/) + Vanilla TS编写

展示文件目录下面（包括子目录）所有的图片，右键可以打开图片（或者双击打开图片），或者跳转到图片目录中去，也可以之间使用取色器去图片中的某个色块
